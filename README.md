# Analise-de-desenhos-tecnicos-Wilson-Sons
Projeto elaborado como desafio de final de curso de Desenvolvedores em IA. Automação que lê desenhos técnicos de construção naval (PDF ou imagem), extrai a lista de materiais usando Inteligência Artificial e compara com o estoque do almoxarifado, notificando o resultado por e-mail e em tempo real no site do projeto.
